def get_code(user, slug, theme, default_tab, iframe_class):

    #print(default_tab)
    if iframe_class != "":
        iframe_class = f' class="{iframe_class}"'
    else:
        iframe_class = ""

    outputCodeText.delete('1.0', END)
    outputCodeText.insert(END, code)

Codepen Embeder

Problema: Dificuldade em personalizar aspeto responsivo de projetos de codepen.com embebidos em iframes para a realização de um portefólio digital.

Solução: Criação de um pequeno utilitário que gera sintaxe personalizada em html otimizada para design responsivo.

Breve descrição: Este programa conta com uma interface desenvolvida em TKinter com Python de modo a permitir ao utilizador digitar o slug do projeto desejado (obtido através do URL do projeto em codepen.com) e obter como resultado uma iframe em html que se adapta automaticamente ao parent element.
Outras funcionalidades: classe personalizada, tema personalizado, e vista predefinida.

Linguagens utilizadas: Python 3

Email Filter

Problema: Necessidade de enviar centenas de emails publicitários para agências de viagens e escolas com uma limitação de até 100 emails por dia imposta pelo provedor de email.

Solução: Automatizar o processo com um script de modo a poupar tempo e trabalho desnecessário.

Breve descrição: O script aceita como input um ficheiro de texto com uma lista desorganizada de emails extraída de uma base de dados em Excel e efetua as seguintes operações:

  1. Validação de emails
  2. Remoção de emails duplicados
  3. Distribuição em grupos de 100
  4. Formatação adequada para envio (separar cada email com um ponto e vírgula)

O output consiste num ficheiro de texto formatado com as condições anteriores.

Linguagens utilizadas: Python 2
